# golang-gin-web-api-template

## Install dependencies

```bash
go mod download
```

## Recommend install make

[Make for Windows](https://gnuwin32.sourceforge.net/packages/make.htm)

## Run the project

```bash
make run

## Auto reload when code change
air
```

## Format the project code

```bash
make format
```

## Build the project

```bash
make build
```
